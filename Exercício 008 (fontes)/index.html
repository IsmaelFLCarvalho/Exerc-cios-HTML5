Principais formatações

Negrito / Destaque

Nessa frase temos um termo em Negrito usando a tag <b> , entranto essa tag não tem semântica, ela só tem forma.

Nessa frase em destaque ela é forte. Logo tem semântica.

Itálico / Ênfase

Nesta frase iremos usar um termo em ITÁLICO usando a tag ("SELECT TEXT" + CTRL + P) após select (i) e depois enter.

Usando agora o texto em ênfase , usa a tag <em> ( semântica )
